# 更新日志

## [1.0.0] - 2026-02-14

### 新增功能
- 弹性工时计算
  - 上班时间：8:00-9:30
  - 对应下班时间：18:00-19:30
  - 早于8:00的上班时间自动调整为8:00

- 工时累加功能
  - 本周超出的工时可累加到后续天数
  - 实时计算剩余工时

- 午休自动扣除
  - 固定午休时间：12:00-14:00（2小时）
  - 跨越午休自动扣除2小时

- 加班时长扣除
  - 支持输入已申请的加班时长
  - 从当天有效工时中扣除

- 日期范围筛选
  - 通过日期选择器选择统计范围
  - 最多支持7天范围筛选
  - 仅统计筛选范围内的工时数据

- 假期同步功能
  - 从在线API获取法定节假日数据
  - 每月只能同步一次
  - 同步后数据保存到本地缓存

- 假期日历显示
  - 法定节假日以淡绿色（#d9f7be）标记
  - 周末以浅绿色（#f6ffed）标记
  - 鼠标悬停显示假期名称

- 版本更新日志
  - 查看软件版本历史
  - 显示功能新增、问题修复、优化改进

- 数据导出
  - 保存数据到本地IndexedDB
  - 导出考勤记录为PNG图片

### 优化改进
- 实时计算工时和下班时间
- 工时进度条可视化展示
- 自动提示异常数据
  - 上班时间早于8:00自动调整
  - 下班时间早于18:00警告
  - 下班时间早于上班时间警告
- 日期范围筛选器UI优化

### 规则限制
- 上班时间最早8:00（自动调整）
- 下班时间最早18:00
- 下班时间必须晚于上班时间
- 日期范围筛选最多7天
- 每月只能同步一次假期数据

### 技术实现
- 前端框架：React 18 + TypeScript
- UI组件库：Ant Design 5.x
- 状态管理：Zustand
- 数据存储：IndexedDB (Dexie.js)
- 时间处理：Day.js
- 图片导出：html2canvas
